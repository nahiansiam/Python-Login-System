<h2>📖 Overview</h2>

<p>
The <strong>Python Secure Login Portal</strong> is a console-based authentication
project built with Python. It allows users to enter a username and password,
verifies credentials, limits the number of failed login attempts, and activates
a security lock after too many unsuccessful tries.
</p>

<p>
This project is ideal for beginners who want to learn:
</p>

<ul>
  <li>Python <strong>while loops</strong></li>
  <li><strong>Conditional statements</strong> (if/else)</li>
  <li>User input handling</li>
  <li>Authentication logic</li>
  <li>Attempt tracking and security concepts</li>
</ul>

<hr>

<h2>✨ Features</h2>

<ul>
  <li>🔐 Username and Password Authentication</li>
  <li>🎯 Secure Credential Verification</li>
  <li>🔄 Limited Login Attempts (5 Attempts)</li>
  <li>⚠️ Failed Login Notifications</li>
  <li>📊 Remaining Attempts Counter</li>
  <li>🚫 Automatic Security Lock After Maximum Failures</li>
  <li>🎉 Successful Login Welcome Message</li>
  <li>🖥️ Simple Console-Based Interface</li>
</ul>

<h2>🔒 Security Workflow</h2>

<ol>
  <li>User enters a username and password.</li>
  <li>Program checks whether the credentials are correct.</li>
  <li>If valid, access is granted.</li>
  <li>If invalid, an error message is displayed.</li>
  <li>Remaining attempts are shown.</li>
  <li>After 5 failed attempts, the account is locked.</li>
</ol>
